# TBTX Full-Stack Ecosystem

This is the production-ready, full-stack web ecosystem for TransformBy10X (TBTX), BizBuilders AI (BBAI), and BizBot Marketing (BBM).

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Components**: [Lucide React](https://lucide.dev/)

## Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Locally
```bash
npm run dev
```

The site will be available at [http://localhost:3000](http://localhost:3000).

## Directory Structure
- `src/app/`: Contains the routes and main page logic.
  - `tbtx/`: TransformBy10X (Philosophy / Hard Cuts)
  - `bbai/`: BizBuilders AI (Infrastructure / System Reveal)
  - `bbm/`: BizBot Marketing (Live Diagnosis Surface)
  - `diagnostic/blueprint/`: AI Blueprint Result Page
- `src/config/`: Contains the intake engine configuration and question logic.
- `src/app/globals.css`: Global styling and design system tokens.

## Design Philosophy
This project follows a **Swiss Industrial Print** aesthetic:
- Rigid grid layouts.
- Zero border radius.
- Macro-typography.
- High-contrast, severe color palettes.
