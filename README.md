# TBTX Full-Stack Ecosystem

This is the production-ready, full-stack web ecosystem for TransformBy10X (TBTX), BizBuilders AI (BBAI), and BizBot Marketing (BBM).

Aligned to the full TransformBy10X Ecosystem Onboarding Brief and positioning (WIN / GOAL / FLOW hierarchy, exact sample messaging from section 29, execution standards, and master sequence).

## Hierarchy
WIN (decide) → GOAL (Governed Operational Architecture Logic) → FLOW (Frictionless Leveraged Orchestrated Workflow)

FAAS is the runtime implementation of GOAL + FLOW.

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Components**: [Lucide React](https://lucide.dev/)

## Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Locally
```bash
npm run dev
```

The site will be available at [http://localhost:3000](http://localhost:3000).

## Directory Structure
- `src/app/`: Contains the routes and main page logic.
  - `tbtx/`: TransformBy10X (Philosophy / Hard Cuts)
  - `bbai/`: BizBuilders AI (Infrastructure / System Reveal)
  - `bbm/`: BizBot Marketing (Live Diagnosis Surface)
  - `diagnostic/`: Context Architecture Assessment + Blueprint
- `src/config/`: Contains the intake engine configuration and question logic.
- `src/app/globals.css`: Global styling and design system tokens.

This project follows a **Swiss Industrial Print** aesthetic:
- Rigid grid layouts.
- Zero border radius.
- Macro-typography.
- High-contrast, severe color palettes.

All assets follow the 20 Execution Standards and Hosting AI Operating Instructions: correct brands, no invented offers, every asset routes to a funnel/CTA/revenue path, WIN memory capture considered, Context Architecture first.
